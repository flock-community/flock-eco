# Flock ecosystem

Flock. eco is a collection of fullstack moduels which bootstraps development of platforms which can be deployed in the cloud.

## Setup
```
Install Java 11
Install Maven
Install Nvm
```

## Install
```
mvn clean install
nvm use
npm install
npm run bootstrap
```

## start example app
```
./start-example-kotlin.sh
./start-example-react.sh
```

## Publish
```
./eco-scripts/prepare.sh *.*.*
```
